ST_GeomFromGeoJSON¶

arctern._wrapper_func.ST_GeomFromGeoJSON(json)[源代码]¶

Construct geometry from the GeoJSON representation.

参数

json (Series(dtype: object)) – Geometries in json format.

返回类型

Series(dtype: object)

返回

Geometries in WKB form.

Example
>>> import pandas
>>> import arctern
>>> data = pandas.Series(["{"type":"LineString","coordinates":[[1,2],[4,5],[7,8]]}"])
>>> string_ptr = arctern.ST_AsText(arctern.ST_GeomFromGeoJSON(data))
>>> print(string_ptr)
    0    LineString (1 2,4 5,7 8)
    dtype: object
