欢迎访问 Arctern 文档¶

Arctern 聚焦时空数据科学，目标是加速时空数据的处理、分析、模型预测与呈现。Arctern 基于开源大数据生态，构建灵活、强大、高性能的时空数据分析平台，帮助你应对 5G/IoT 带来的新型数据挑战。

如果你正在寻找下述问题的解决方案，欢迎使用 Arctern：

  • 高效的标准地理信息分析库（OGC标准）。

  • 高级时空数据处理算法与模型，如轨迹分析、电子围栏数据处理、时空数据统计、时空数据挖掘等。

  • 海量时空数据的渲染与可视化。

  • 与 Pandas 生态无缝集成的时空数据分析。

  • GPU 加速的地理信息处理。
