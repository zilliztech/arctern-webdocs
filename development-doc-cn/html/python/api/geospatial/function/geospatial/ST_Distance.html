ST_Distance¶

arctern.ST_Distance(geo1, geo2)[源代码]¶

计算几何体 geo1 与几何体 geo2 之间的平面距离。

参数

  • geo1 (Series(dtype: object)) – WKB 格式的几何体。

  • geo2 (Series(dtype: object)) – WKB 格式的几何体。

返回类型

Series(dtype: float64)

返回

geo1 与 geo2 之间的平面距离的值。

例子
>>> import pandas
>>> import arctern
>>> p11 = "LINESTRING(9 0,9 2)"
>>> p12 = "POINT(10 2)"
>>> data1 = pandas.Series([p11, p12])
>>> p21 = "POLYGON((0 0,0 8,8 8,8 0,0 0))"
>>> p22 = "POLYGON((0 0,0 8,8 8,8 0,0 0))"
>>> data2 = pandas.Series([p21, p22])
>>> rst = arctern.ST_Distance(arctern.ST_GeomFromText(data2), arctern.ST_GeomFromText(data1))
>>> print(rst)
    0    1.0
    1    2.0
    dtype: float64
