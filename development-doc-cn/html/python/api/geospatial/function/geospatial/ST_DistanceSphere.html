ST_DistanceSphere¶

arctern.ST_DistanceSphere(geo1, geo2)[源代码]¶

根据经纬度坐标，计算地球表面两点之间的球面距离（单位为米）。

参数

  • geo1 (Series(dtype: object)) – WKB 格式的几何体。

  • geo2 (Series(dtype: object)) – WKB 格式的几何体。

返回类型

Series(dtype: float64)

返回

两点之间球面距离的值。

例子
>>> import pandas
>>> import arctern
>>> p11 = "POINT(10 2)"
>>> p12 = "POINT(10 2)"
>>> data1 = pandas.Series([p11, p12])
>>> p21 = "POINT(10 2)"
>>> p22 = "POINT(10 3)"
>>> data2 = pandas.Series([p21, p22])
>>> rst = arctern.ST_DistanceSphere(arctern.ST_GeomFromText(data2), arctern.ST_GeomFromText(data1))
>>> print(rst)
    0         1.0
    1    111226.3
    dtype: float64
