ST_Length¶

arctern.ST_Length(geos)[源代码]¶

计算线性几何体的长度。

参数

geos (Series(dtype: object)) – WKB 格式的几何体。

返回类型

Series(dtype: float64)

返回

几何体的长度。

例子
>>> import pandas
>>> import arctern
>>> data = ["LINESTRING(0 0,0 1)", "LINESTRING(1 1,1 4)"]
>>> data = pandas.Series(data)
>>> rst = arctern.ST_Length(arctern.ST_GeomFromText(data))
>>> print(rst)
    0    1.0
    1    3.0
    dtype: float64
