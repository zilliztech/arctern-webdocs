plot¶

arctern.plot(ax, geoms, **style_kwds)[源代码]¶

绘制一组几何体。参数“linewidth”，“linestyle”，“edgecolor”,”facecolor”，“color”,“marker”，“markersize”用来描述绘制图形的样式。

对于几何体“Polygon”，“MultiPolygon”只需要配置“linewidth”，“edgecolor”，“facecolor”，“linestyle”四个参数。

对于几何体“Linestring”,“MultiLinestring”只需要配置“color”，“linewidth”，“linestyle”三个参数。

对于几何体“Point”，“MultiPoint”，只需要配置“color”，“marker”，“markersize”三个参数。

参数

  • ax (matplotlib.axes.Axes) – 用来绘制几何体的坐标轴。

  • geoms (Series or DataFrame) – 几何体序列。

  • linewidth (list(float)) – 线的宽度，默认值是1.0。

  • linestyle (list(string)) – 线的风格，默认值是‘-’。

  • edgecolor (list(string)) – 几何体边的颜色，默认值是‘black’。

  • facecolor (list(string)) – 几何体面的颜色，默认值时‘C0’。

  • color (list(string)) – 几何体的颜色，默认值是‘C0’。

  • marker (string) – 点的形状，默认值是‘o’。

  • markersize (double) – 点的大小，默认值是6.0。

例子
>>> import pandas
>>> import matplotlib.pyplot as plt
>>> import arctern
>>> raw_data = []
>>> raw_data.append('point(0 0)')
>>> raw_data.append('linestring(0 10, 5 5, 10 0)')
>>> raw_data.append('polygon((2 2,2 3,3 3,3 2,2 2))')
>>> raw_data.append("GEOMETRYCOLLECTION("
                    "polygon((1 1,1 2,2 2,2 1,1 1)),"
                    "linestring(0 1, 5 6, 10 11),"
                    "POINT(4 7))")
>>> arr_wkt = pandas.Series(raw_data)
>>> arr_wkb = arctern.ST_CurveToLine(arctern.ST_GeomFromText(arr_wkt))
>>> df = pandas.DataFrame({'wkb':arr_wkb})
>>> fig, ax = plt.subplots()
>>> arctern.plot(ax, df,
                 color=['orange', 'green', 'blue', 'red'],
                 marker='^',
                 markersize=100,
                 linewidth=[None, 7, 8, 5],
                 linestyle=[None, 'dashed', 'dashdot', None],
                 edgecolor=[None, None, 'red', None],
                 facecolor=[None, None, 'black', None])
>>> ax.grid()
>>> fig.savefig('/tmp/plot_test.png')
