arctern.plot.fishnetmap¶

arctern.plot.fishnetmap(ax, points, weights, bounding_box, color_gradient=['#0000FF', '#FF0000'], cell_size=4, cell_spacing=1, opacity=1.0, coordinate_system='epsg:3857', aggregation_type='sum', **extra_contextily_params)[源代码]¶

在 Matplotlib 中绘制渔网图。

参数

  • ax (matplotlib.axes.Axes) – 绘制几何图形的画布的坐标轴。

  • points (GeoSeries) – 一组点。

  • weights (Series) – 多边形的颜色权重。

  • bounding_box (list) – 地图的边界，如 [west, south, east, north]。

  • color_gradient (list, optional) – 颜色梯度，默认值为 [“#0000FF”, “#FF0000”]。你可以使用 [“hex_color”] 来为所有几何体指定相同的颜色，或者使用 [“hex_color1”, “hex_color2”] 来指定从 “hex_color1” 到 “hex_color2” 的渐变色。

  • cell_size (int, optional) – 网格的边长，默认值为 4.

  • cell_spacing (int, optional) – 相邻网格的边距，默认值为 1。

  • opacity (float, optional) – 渔网的不透明度，范围是 0.0 到 1.0，默认值为 1.0。

  • coordinate_system (str, optional) – 为所有几何图形设置 CRS（Coordinate Reference System，坐标参考系），默认值为 ‘EPSG:3857’。目前仅支持 SRID（Spatial Reference Identifier，空间参考标识符）作为 CRS 的 WKT 表示。

  • aggregation_type (str, optional) – 聚合类型，默认值为 ‘sum’。

  • **extra_contextily_params (dict) – 传到 contextily.add_basemap 中的其他参数。

实际案例

>>> import pandas as pd
>>> import numpy as np
>>> import arctern
>>> import matplotlib.pyplot as plt
>>> # read from test_data.csv
>>> # Download link: https://raw.githubusercontent.com/arctern-io/arctern-resources/benchmarks/benchmarks/dataset/layer_rendering_test_data/test_data.csv
>>> df = pd.read_csv("/path/to/test_data.csv", dtype={'longitude':np.float64, 'latitude':np.float64, 'color_weights':np.float64, 'size_weights':np.float64, 'region_boundaries':np.object})
>>> points = arctern.GeoSeries.point(df['longitude'], df['latitude'])
>>> # render fishnet
>>> fig, ax = plt.subplots(figsize=(10, 6), dpi=200)
>>> arctern.plot.fishnetmap(ax, points=points, weights=df['color_weights'], bounding_box=[-74.01424568752932, 40.72759334104623, -73.96056823889673, 40.76721122683304], cell_size=8, cell_spacing=2, opacity=1.0, coordinate_system="EPSG:4326")
>>> plt.show()
