arctern.plot.plot_geometry¶

arctern.plot.plot_geometry(ax, geoms, **style_kwds)[源代码]¶

Plot a collection of geometries to ax. Parameters ‘linewidth’, ‘linestyle’, ‘edgecolor’, ‘facecolor’, ‘color’, ‘marker’, ‘markersize’ are used to describe the style of plotted figure.

For geometry types Polygon and MultiPolygon, only ‘linewidth’, ‘linestyle’, ‘edgecolor’, ‘facecolor’ are effective.

For geometry types Linestring and MultiLinestring, only ‘color’, ‘linewidth’, ‘linestyle’ are effective.

For geometry types Point and MultiPoint, only ‘color’, ‘marker’, ‘markersize’ are effective.

参数

  • ax (matplotlib.axes.Axes) – The axes where geometries will be plotted.

  • geoms (Series or DataFrame) – sequence of geometries.

  • linewidth (list(float)) – The width of line, the default value is 1.0.

  • linestyle (list(string)) – The style of the line， the default value is ‘-‘.

  • edgecolor (list(string)) – The edge color of the geometry, the default value is ‘black’.

  • facecolor (list(string)) – The color of the face of the geometry, the default value is ‘C0’.

  • color (list(string)) – The color of the geometry, the default value is ‘C0’.

  • marker (string) – The shape of point, the default value is ‘o’.

  • markersize (double) – The size of points, the default value is 6.0.

  • alpha (double) – The transparency of the geometry, the default value is 1.0.

Example
>>> import pandas
>>> import matplotlib.pyplot as plt
>>> import arctern
>>> raw_data = []
>>> raw_data.append('point(0 0)')
>>> raw_data.append('linestring(0 10, 5 5, 10 0)')
>>> raw_data.append('polygon((2 2,2 3,3 3,3 2,2 2))')
>>> raw_data.append("GEOMETRYCOLLECTION("
...                 "polygon((1 1,1 2,2 2,2 1,1 1)),"
...                 "linestring(0 1, 5 6, 10 11),"
...                 "POINT(4 7))")
>>> arr_wkt = pandas.Series(raw_data)
>>> arr_wkb = (arctern.GeoSeries(arr_wkt)).curve_to_line()
>>> df = pandas.DataFrame({'wkb':arr_wkb})
>>> fig, ax = plt.subplots()
>>> arctern.plot.plot_geometry(ax, df,
...                            color=['orange', 'green', 'blue', 'red'],
...                            marker='^',
...                            markersize=100,
...                            linewidth=[None, 7, 8, 5],
...                            linestyle=[None, 'dashed', 'dashdot', None],
...                            edgecolor=[None, None, 'red', None],
...                            facecolor=[None, None, 'black', None])
>>> ax.grid()
>>> fig.savefig('/tmp/plot_test.png')
