GeoSeries¶

GeoSeries 是通过扩展 Pandas Series 来存储和处理几何体的 Series

Constructor¶

GeoSeries([data, index, name, crs])

存储 WKB 格式的字节对象几何体的 Series。

Attributes¶

GeoSeries.is_valid

判断每个几何体是否有效。

GeoSeries.length

计算每个几何体的长度

GeoSeries.is_simple

判断每个几何体是否简单。

GeoSeries.area

计算几何体的二维笛卡尔（平面）面积。

GeoSeries.geom_type

For each geometry in geometries, return a string that indicates is type.

GeoSeries.centroid

计算每个几何体的中心点

GeoSeries.convex_hull

对于 GeoSeries 对象中的每个几何体，计算包围它的最小的凸几何体

GeoSeries.npoints

对于 GeoSeries 对象中的每个几何体，计算它的点的数量。

GeoSeries.envelope

对于 GeoSeries 对象中的每个几何体，计算它的最小矩形边界范围。

Constructing Geometry¶

GeoSeries.point(x, y[, crs])

根据坐标集合构造 Point 几何体。

GeoSeries.polygon_from_envelope(min_x, …)

对于 GeoSeries 对象中的每个几何体，根据给定的参数计算它的最小矩形边界范围，且该矩形的边与坐标轴平行。

GeoSeries.geom_from_geojson(json[, crs])

根据 GeoJSON 代表字符串构造几何体。

GeoSeries.as_geojson()

将几何体转换为 GeoJSON 格式字符串。

GeoSeries.to_wkt()

将几何体转换为 WKT 格式字符串。

GeoSeries.to_wkb()

将几何体转换为 WKB 格式字符串。

Processing Geometry¶

GeoSeries.buffer(distance)

对于 GeoSeries 对象中的每个几何体，创建一个与它的最远距离不大于 distance 的几何体。

GeoSeries.precision_reduce(precision)

对于 GeoSeries 对象中的每个几何体，根据指定的有效数字位数 precision 创建降低坐标精度后的几何体。

GeoSeries.intersection(other)

对于 GeoSeries 对象中的每个几何体，计算它与 other 对象中相同位置的几何体之间交集。

GeoSeries.make_valid()

对于 GeoSeries 对象中的每个几何体，根据它创建一个新的有效的几何体。在构造新几何体过程中，不会删除原始几何体的任何顶点。如果原始几何体本来就是有效的，则直接返回原始几何体。

GeoSeries.simplify(tolerance)

Returns a “simplified” version for each geometry using the Douglas-Peucker algorithm.

GeoSeries.set_crs(crs)

设置 GeoSeries 的坐标系。

GeoSeries.crs

Return the coordinate system of the GeoSeries.

GeoSeries.to_crs(crs)

转换几何体的 坐标参考系。

GeoSeries.curve_to_line()

对于 GeoSeries 对象中的每个几何体，计算它的近似表示。近似表示的方法是将每个几何图形中的曲线转换为近似线性表示。

Spatial Relationship¶

GeoSeries.geom_equals(other)

对于 GeoSeries 对象中的每个几何体，判断它是否与 other 对象中相同位置的几何体等价。“等价”表示两个几何体的几何结构相同。

GeoSeries.touches(other)

对于 GeoSeries 对象中的每个几何体，判断它是否与 other 对象中相同位置的几何体相邻。“相邻”表示两个几何体在边界上有共同的点。

GeoSeries.overlaps(other)

对于 GeoSeries 对象中的每个几何体，判断它是否与 other 对象中相同位置的几何体重叠。“重叠”表示两个几何体相交且不互相包含。

GeoSeries.crosses(other)

Check whether each geometry and other(elementwise) “spatially cross”.

GeoSeries.contains(other)

对于 GeoSeries 对象中的每个几何体，判断它是否包含 other 对象中相同位置的几何体。

GeoSeries.intersects(other)

Check whether each geometry intersects other (elementwise).

GeoSeries.within(other)

对于 GeoSeries 对象中的每个几何体，判断它是否在 other 对象中相同位置的几何体的内部。

Measurement¶

GeoSeries.distance_sphere(other)

返回两个经纬度点之间的以米为单位的距离。

GeoSeries.distance(other)

计算两个几何体之间的最小二维笛卡尔（平面）距离。

GeoSeries.hausdorff_distance(other)

Returns the Hausdorff distance between each geometry and other.

Aggregation¶

GeoSeries.unary_union()

Return a geometry that represents the union of all geometries in the GeoSeries.

GeoSeries.envelope_aggr()

对于 GeoSeries 对象中的所有几何体集合，计算它的最小矩形边界范围。

Pandas Methods¶

GeoSeries.isna()

检测丢失值

GeoSeries.notna()

检测未丢失值

GeoSeries.fillna([value, method, axis, …])

用几何体填补 NA 值，其可为 WKT 或 WKB 格式。

GeoSeries.equals(other)

测试两个几何体是否含有相同元素。
