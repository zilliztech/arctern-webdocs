arctern.GeoSeries¶

class arctern.GeoSeries(data=None, index=None, name=None, crs=None, **kwargs)[源代码]¶

Series 存储几何体，此几何体是 WKB 格式的字节对象。

参数

  • data (array-like, Iterable, dict, or scalar value(str or bytes)) – 要存储的几何体，可以是 WKB 格式的字节或 WKT 格式的字符串。

  • index (array-like or Index (1d)) – 与 pandas Series 相同。值必须是可哈希的，并且与 data 的长度相同。允许使用非唯一索引值。如果未提供，则默认为 RangeIndex（0，1，2，…，n）。如果同时使用了字典和索引序列，则索引将覆盖在字典中找到的键。

  • name (str, optional) – 赋给 Series 的名称

  • crs (str, optional) – GeoSeries 的参考坐标系现在只支持 SRID 格式。

  • kwargs – 传到 GeoSeries 构造器中的其他参数，例如 copy。

例子

>>> from arctern import GeoSeries
>>> s = GeoSeries(["POINT(1 1)", "POINT(1 2)"])
>>> s
0    POINT (1 1)
1    POINT (1 2)
dtype: GeoDtype

方法

as_geojson()

将每个几何体转换为 GeoJSON 格式的字符串。

buffer(distance)

对于 GeoSeries 对象中的每个几何体，创建一个与它的最远距离不大于 distance 的几何体。

contains(other)

对于 GeoSeries 对象中的每个几何体，判断它是否包含 other 对象中相同位置的几何体。

crosses(other)

对于 GeoSeries 对象中的每个几何体，判断它是否与 other 对象中相同位置的几何体相交。

curve_to_line()

对于 GeoSeries 对象中的每个几何体，计算它的近似表示。近似表示的方法是将每个几何图形中的曲线转换为近似线性表示。

distance(other)

对于 GeoSeries 对象中的每个几何体，判断它与 other 对象中相同位置的几何体之间的最短二维笛卡尔（平面）距离。

distance_sphere(other)

对于 GeoSeries 对象中的每个几何体，根据经纬度坐标计算地球表面两点之间的最短球面距离。该方法使用 SRID 定义的地球和半径。

envelope_aggr()

计算包含一组几何体的最小矩形边界范围，且该矩形的边与坐标轴平行。

equals(other)

判断两个几何体是否含有相同元素。

fillna([value, method, axis, inplace, …])

用几何体填补 NA 值，该几何体可以为 WKT 或 WKB 格式。

from_geopandas(data)

由 geopandas GeoSeries 构造几何体。

geom_equals(other)

对于 GeoSeries 对象中的每个几何体，判断它是否与 other 对象中相同位置的几何体等价。“等价”表示两个几何体的几何结构相同。

geom_from_geojson(json[, crs])

根据 GeoJson 字符串构造几何体。

hausdorff_distance(other)

对于 GeoSeries 对象中的每个几何体，判断它与 other 对象中相同位置的几何体之间的 Hausdorff 距离。此距离用于度量两个几何体之间的相似度。

intersection(other)

对于 GeoSeries 对象中的每个几何体，计算它与 other 对象中相同位置的几何体之间交集。

intersects(other)

对于 GeoSeries 对象中的每个几何体，判断它是否与 other 对象中相同位置的几何体存在交集。

isna()

检测丢失值。

对于 GeoSeries 对象中的每个几何体，根据它创建一个新的有效的几何体。在构造新几何体过程中，不会删除原始几何体的任何顶点。如果原始几何体本来就是有效的，则直接返回原始几何体。

Create a valid representation of each geometry without losing any of the input vertices.

notna()

检测未丢失值。

overlaps(other)

对于 GeoSeries 对象中的每个几何体，判断它是否与 other 对象中相同位置的几何体重叠。“重叠”表示两个几何体相交且不互相包含。

根据横坐标集合 x 和纵坐标集合 y 构造一组 POINT 对象。

Construct Point geometries according to the coordinates.

polygon_from_envelope(min_x, min_y, max_x, max_y)

对于 GeoSeries 对象中的每个几何体，根据给定的参数计算它的最小矩形边界范围，且该矩形的边与坐标轴平行。

precision_reduce(precision)

对于 GeoSeries 对象中的每个几何体，根据指定的有效数字位数 precision 创建降低坐标精度后的几何体。

set_crs(crs)

给 GeoSeries 设置坐标系。

simplify(tolerance)

对于 GeoSeries 对象中的每个几何体，计算它的简化表示。简化表示的算法是 Douglas-Peucker 算法。

to_crs(crs)

转换几何体的坐标参考系。

to_geopandas()

将 Arctern GeoSeries 转换为 GeoPandas GeoSeries。

to_wkb()

将几何体转换为 WKB 格式的字节对象。

to_wkt()

将几何体转换为 WKT 格式的字符串。

touches(other)

对于 GeoSeries 对象中的每个几何体，判断它是否与 other 对象中相同位置的几何体相邻。“相邻”表示两个几何体在边界上有共同的点。

unary_union()

返回一个代表一组几何体的并集的几何体。

within(other)

对于 GeoSeries 对象中的每个几何体，判断它是否在 other 对象中相同位置的几何体的内部。

Attributes

area

计算 GeoSeries 对象中每个几何体的面积。

centroid

计算 GeoSeries 对象中每个几何体的中心点。

convex_hull

对于 GeoSeries 对象中的每个几何体，计算包围它的最小的凸几何体。

crs

Return the coordinate system of the GeoSeries.

envelope

对于 GeoSeries 对象中的每个几何体，计算它的最小矩形边界范围。

geom_type

对于 GeoSeries 对象中的每个几何体，返回一个表示类型的字符串。

is_simple

判断 GeoSeries 对象中的每个几何体是否简单。这里的“简单”表示某个几何图形没有异常的几何图形点，例如自相交或相切。

is_valid

判断 GeoSeries 对象中的每个几何体是否有效。

length

计算 GeoSeries 对象中每个几何体的长度。如果几何体不是 LineString 和 MultiLineString 类型的几何体， 则它的长度为 0 。

npoints

对于 GeoSeries 对象中的每个几何体，计算它的点的数量。
