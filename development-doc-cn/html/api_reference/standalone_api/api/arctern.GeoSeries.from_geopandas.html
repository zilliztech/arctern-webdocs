arctern.GeoSeries.from_geopandas¶

classmethod GeoSeries.from_geopandas(data)[源代码]¶

Construct geometries from geopandas GeoSeries.

Rtype data

geopandas.GeoSeries

参数

data – Source geometries data.

返回类型

arctern.GeoSeries

返回

A arctern.GeoSeries constructed from geopandas.GeoSeries.

Example

>>> import geopandas as gpd
>>> from shapely.geometry import Point, Polygon
>>> from arctern import GeoSeries
>>> gpd_s = gpd.GeoSeries([Point(1,1), Polygon(((1,1), (1,2), (2,3), (1,1)))])
>>> arctern_s = GeoSeries.from_geopandas(gpd_s)
>>> arctern_s
0                    POINT (1 1)
1    POLYGON ((1 1,1 2,2 3,1 1))
dtype: GeoDtype
