arctern.GeoSeries.polygon_from_envelope¶

classmethod GeoSeries.polygon_from_envelope(min_x, min_y, max_x, max_y, crs=None)[源代码]¶

Construct polygon(rectangle) geometries from arr_min_x, arr_min_y, arr_max_x, arr_max_y and special coordinate system. The edges of polygon are parallel to coordinate axis.

参数

  • min_x (Series(dtype: float64)) – The minimum value of x coordinate of the rectangles.

  • min_y (Series(dtype: float64)) – The minimum value of y coordinate of the rectangles.

  • max_x (Series(dtype: float64)) – The maximum value of x coordinate of the rectangles.

  • max_y (Series(dtype: float64)) – The maximum value of y coordinate of the rectangles.

  • crs (string, optional) – Must be SRID format string.

返回类型

GeoSeries

返回

A GeoSeries contains geometries.

Example

>>> from pandas import Series
>>> from arctern import GeoSeries
>>> min_x = Series([0.0, 1.0])
>>> max_x = Series([2.0, 1.5])
>>> min_y = Series([0.0, 1.0])
>>> max_y = Series([1.0, 1.5])
>>> GeoSeries.polygon_from_envelope(min_x, min_y, max_x, max_y)
0                POLYGON ((0 0,0 1,2 1,2 0,0 0))
1    POLYGON ((1 1,1.0 1.5,1.5 1.5,1.5 1.0,1 1))
dtype: GeoDtype
