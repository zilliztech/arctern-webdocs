arctern.GeoSeries.geometry_type¶

property GeoSeries.geometry_type¶

对输入的每一个几何体，返回表示其类型的字符串。

返回类型

Series(dtype: object)

返回

几何体的类型，例如 ST_LINESTRING，ST_POLYGON，ST_POINT，ST_MULTIPOINT。

例子

>>> from arctern import GeoSeries
>>> s = GeoSeries(["POINT(1 1)", "POLYGON ((1 1, 3 1, 3 3, 1 3, 1 1))"])
>>> s.geometry_type
0      ST_POINT
1    ST_POLYGON
dtype: object
