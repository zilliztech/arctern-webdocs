arctern.GeoSeries.distance_sphere¶

GeoSeries.distance_sphere(other)[源代码]¶

计算地球表面两点之间的最短球面距离（单位为米）。

使用 SRID 定义的地球和半径。只有’EPSG:4326’能够计算球面距离。

参数

other (scalar bytes object geometry or GeoSeries) – 用来计算每两个几何体之间的球面距离的几何体。它可以是标量 WKB 格式的字节对象或 GeoSeries。如果另一个是标量字节对象，我们假定另一个的坐标参考系是’EPSG:4326’。

返回类型

Series(dtype: float64)

返回

Series 包含每个几何体与其他几何体之间的球面距离。

例子

>>> from arctern import GeoSeries
>>> s1 = GeoSeries(["POINT(10 2)"], crs="EPSG:4326")
>>> s2 = GeoSeries(["POINT(10 3)"], crs="EPSG:4326")
>>> s2.distance_sphere(s1)
0    111226.3
dtype: float64
