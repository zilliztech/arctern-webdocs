arctern.nearest_road¶

arctern.nearest_road(roads, points)[源代码]¶

在 roads 中的所有道路上查找最接近 points 中的点的道路。points 中的点不必都位于连续路径上。

参数

  • roads (Series) – 一组 WKB 格式的 LINGSTRING 对象。

  • points (Series) – 一组 WKB 格式的 POINT 对象。

返回

一个 WKB 格式的 POINT 对象。

返回类型

Series

实际案例

>>> import arctern
>>> data1 = arctern.GeoSeries(["LINESTRING (1 2,1 3)"])
>>> data2 = arctern.GeoSeries(["POINT (1.001 2.5)"])
>>> rst = arctern.GeoSeries(arctern.nearest_road(data1, data2)).to_wkt()
>>> rst
    0    LINESTRING (1 2,1 3)
    dtype: object
