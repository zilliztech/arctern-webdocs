arctern.within_which¶

arctern.within_which(left, right)[源代码]¶

对于 left 中的每个几何体，在 right 中搜索包含该几何体的几何体。

参数

  • left (GeoSeries) – 几何体序列。

  • right (GeoSeries) – 几何体序列。

返回

在 right 中的几何体索引。例如，返回的 Series 中，索引为 i、值为 j 的元素表示几何体 left[i] 在几何体 right[j] 之内。

返回类型

Series

实际案例

>>> from arctern import *
>>> data1 = GeoSeries(["Point(0 0)", "Point(1000 1000)", "Point(10 10)"])
>>> data2 = GeoSeries(["Polygon((9 10, 11 12, 11 8, 9 10))", "Polygon((-1 0, 1 2, 1 -2, -1 0))"])
>>> res = within_which(data1, data2)
>>> print(res)
    0    1
    1    <NA>
    2    0
    dtype: object
