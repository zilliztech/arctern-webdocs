ST_Buffer¶

arctern.ST_Buffer(geos, distance)[源代码]¶

返回一个与给定的几何体的距离不大于“distance”的几何体。

参数

  • geos (Series(dtype: object)) – WKB 格式的几何体。

  • distance (float) – 返回的几何体与给定的几何体最大距离。

返回类型

Series(dtype: object)

返回

WKB 格式的几何体。

例子
>>> import pandas
>>> import arctern
>>> data = ["POINT (0 1)"]
>>> data = pandas.Series(data)
>>> rst = arctern.ST_AsText(arctern.ST_Buffer(arctern.ST_GeomFromText(data), 0))
>>> print(rst)
    0    POLYGON EMPTY
    dtype: object
