ST_Centroid¶

arctern._wrapper_func.ST_Centroid(geos)[源代码]¶

Compute the centroid of geometry.

参数

geos (Series(dtype: object)) – Geometries in WKB form.

返回类型

Series(dtype: object)

返回

The centroid of geometry in WKB form..

Example
>>> import pandas
>>> import arctern
>>> data = ["POLYGON((0 0,1 0,1 1,0 1,0 0))", "POLYGON((0 0,0 8,8 8,8 0,0 0))"]
>>> data = pandas.Series(data)
>>> rst = arctern.ST_AsText(arctern.ST_Centroid(arctern.ST_GeomFromText(data)))
>>> print(rst)
    0    POINT (0.5 0.5)
    1    POINT (4 4)
    dtype: object
