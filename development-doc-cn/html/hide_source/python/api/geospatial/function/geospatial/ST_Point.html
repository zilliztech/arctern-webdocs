ST_Point¶

arctern._wrapper_func.ST_Point(x, y)[源代码]¶

Construct Point geometries according to the coordinates.

参数

  • x (Series(dtype: float64)) – Abscissa of the point.

  • y (Series(dtype: float64)) – Ordinate of the point.

返回类型

Series(dtype: object)

返回

Point in WKB form.

Example
>>> import pandas
>>> import arctern
>>> data1 = pandas.Series([1.3, 2.5])
>>> data2 = pandas.Series([1.3, 2.5])
>>> string_ptr = arctern.ST_AsText(arctern.ST_Point(data1, data2))
>>> print(string_ptr)
    0    POINT (1.3 3.8)
    1    POINT (2.5 4.9)
    dtype: object
